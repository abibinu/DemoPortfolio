# DemoPortfolio

A demo portfolio website built with HTML.

## About

This repository contains a simple portfolio website project showcasing web development fundamentals. It serves as a demonstration of HTML structure and layout.

## Technologies Used

- HTML5
- CSS

## Project Structure

The project follows a simple structure with HTML files for different sections of the portfolio website.

## Features

- Clean and simple HTML-based design
- Easy to understand structure
- Responsive layout foundation

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/abibinu/DemoPortfolio.git
```

2. Navigate to the project directory:
```bash
cd DemoPortfolio
```

3. Open the HTML files in your web browser to view the portfolio

## Usage

Simply open the HTML files in any modern web browser. No build process or dependencies are required.
